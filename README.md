## 🔹 Problem 8: Smart Navigation System (BFS, DFS)

### Description

A navigation system that allows users to input a graph (as edges), choose a start and goal node, and visualize path finding. The system dynamically builds the graph and finds a path using BFS and DFS with step-by-step animation.

### Algorithms Used

* Breadth First Search (BFS) — shortest path in unweighted graphs
* Depth First Search (DFS) — non-optimal traversal for comparison

### Execution Steps

1. Open the website link.
2. Click **Problem 8 - Smart Navigation**.
3. Enter edges (e.g., `A-B, B-C, C-D, A-D`).
4. Enter Start and Goal nodes.
5. Click **Run BFS** or **Run DFS**.
6. Observe animation, path, and nodes explored.

### Sample Output

* BFS Path: A → D | Nodes Explored: 2
* DFS Path: A → B → C → D | Nodes Explored: 4

## 🔹 Problem 14: Rule-Based Insurance Claim Decision System

### Description

A rule-based system where users input facts (policy status, document validity, accident reported) and define logical rules. The system evaluates the rules using propositional logic and produces a final decision.

### Algorithms Used

* Rule-Based Inference (Propositional Logic)
* Logical evaluation using AND, OR, NOT conditions

### Execution Steps

1. Open the website link.
2. Click **Problem 14 - Insurance Decision**.
3. Select input facts (true/false).
4. Enter rules (e.g., `IF policy AND docs THEN approve`).
5. Click **Evaluate**.
6. View facts, applied rule, and final decision.

### Sample Output

Facts:

* Policy Active: true
* Documents Valid: true
* Accident Reported: true

Rule Applied:

* IF policy AND docs THEN approve

Final Decision:

* Claim Approved
